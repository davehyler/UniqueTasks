    Notes:

  • You can drag your task to "In Progress" or "Completed" at any time you feel you have achieved the Task and/or Description of the task by the Due Date.
  • You MUST begin a task as "In Progress" BEFORE you can mark it as complete as logically there must be a specified amount of time where this task has been worked on.
  • You can move the task from "Completed" back to "In Progress" at any time you feel there is more work to be done.
  • Once you begin a task it is "In Progress" and logistically and semantically can never be "not started" again.
  • Finally, make sure you COMPLETE your item before you DELETE IT. If you decide to not finish the item, then it is COMPLETED as a task "To Do".
